傻瓜式系统重装法

如果你现在电脑系统崩了、出点小毛病都还要往电脑店里跑的、别急请往下看

当今这社会、对很多人来说工作总离不开电脑的、尤其对于那些没事也喜欢瞎折腾的人来说、系统奔溃也是长发生的事、

所以思前想后这个对多数人都有用、今天就以比较普遍的Windows操作系统来说吧、

因为比如现在在用Linux的人来说、我想这些小技能从碰电脑开始就会的了、但是我猜肯定还有很多人现在连Windows的操作系统怎么重装都不知道的！

坏了就知道去电脑修理店、找他们给你装个系统也要收你20块、其实不是钱的问题、而是跑来跑去麻烦、最关键的是那些电脑店里给你装的系统全是垃圾、捆绑一堆垃圾软件、所以我每次买新电脑或者系统崩了无法恢复、我只装纯净版的原系统！

其实Windows系统重装很简单、只要把启动U盘刻录好、进入pe分好区、基本都是闭着眼睛下一步直到完成就好了！

  • 如果你懒、最简单高效的方法是去淘宝几十块买一个别人已经刻录好的
  • 去淘宝直接关键字《纯净版系统U盘》
  • 买个质量好点的终身受用、从XP32位到win10 32位 到64位 都可以买到
  • 买来的系统U盘也还有安装教程、
  1. 第一步只需要进入blos系统设置开机启动项为U盘启动后、再把blos系统时间调好在退出重启电脑、选择U盘启动就可以直接进入pe系统了
  2. 进入blos系统的方法、再开机的时候不停的按f8 或者f12 每个牌子的电脑进入的方法都不一样、总之就是从ESC.f1到f12中间其中一个、要么慢慢试要么根据你电脑的牌子去问百度
  3. 进入PE后最主要的就是分区、现在的分区工具没有像很多年前的那种一样麻烦、基本都是傻瓜式的、进去找到分区工具后、可以选择快速分区、然后根据你自己的喜好需要给硬盘分成几个盘、在选择重做引导记录、然后保存退出、
  4. 然后返回pe系统直接选择需要的系统只管下一步下一步就好了、装好后如果是纯净版的连上网去下载驱动精灵、或者驱动管家这类的驱动软件把驱动装好就搞定了！
  5. 去淘宝买的启动U盘安装过程大部分都这这样、超级简单！

如果想用我想分享的方式自己制作启动u盘

第一步：
先准备一个大于4G的空闲U盘、把我分享的两个刻录工具下载到自己电脑、建议下载第一个比较便携几乎是傻瓜式的安装好后、看一遍你就知道怎么刻录了、然后再根据你的需求下载相应的系统镜像到本地电脑

至于第二个软碟通貌似是要收费的、但是在安装的时候你可以选择试用、
然后就可以刻录了、把刻录软件打开以后有个选择文件的选项、然后找到你下载的系统镜像、就可以直接刻录了、

如果不知道怎么使用可以去百度搜关键字比如：软碟通使用教程、rufus刻录教程等等、、、、太简单了我就懒得写了

一般Windows系统开机出现蓝屏啦或者出现一堆英文字母啦这些、都可以去blos系统设置、基本都是这里出了问题、如果系统直接系统文件都被损坏了、那就直接重装系统、半个小时就搞定了！

windows 系统 KMS 激活

管理员权限运行 cmd

slmgr /skms kms.v0v.bid && slmgr /ato

或者直接访问：技术员 KMS 激活 bat 一键脚本 点击下载

备用服务器

k01.v0v.bid - k13.v0v.bid

如激活失败使用管理员权限运行 cmd 执行以下命令更换密钥后重新激活

（密钥见：https://v0v.bid/kms.html）

slmgr /ipk 这里根据系统版本填写对应密钥
KMS 批量激活密钥广泛用于机房服务器操作系统和大型企业内部批量激活员工电脑。目前公开的 KMS 服务器封杀力度很大，为了节省大家的时间与精力，请低调使用，且用且珍惜。

KMS 激活有 180 天期限，此期限称为激活有效间隔。若要保持激活状态，您的系统必须通过至少每 180 天连接一次 KMS 服务器来续订激活。

默认情况下，系统每 7 天自动进行一次激活续订尝试。在续订客户端激活之后，激活有效间隔重新开始。

综上所述，只要您的电脑不超过 180 天以上无法连接互联网，就无需进行任何操作，系统会自行续期保持激活状态。即永久激活。

如果需要激活码这类的、office这类的激活码什么的、最简单快速的就是淘宝5块钱一个、如果不想花钱就去百度或者谷歌直接搜、慢慢试、很多还是能用的！就是这么简单！！！！！
